Nützliche APIs

Exchange Rate API

Man kann eine Währung eingeben und erhält eine Liste mit allen Umrechnungskursen für andere Fremdwährungen

https://api.exchangerate-api.com/v4/latest/USD

Random User

Zum Testen von Logins werden User erstellt

https://randomuser.me/api

JSON Placeholder

Es stehen Posts, Users etc. zur Verfügung um Applikationen zu testen

https://jsonplaceholder.typicode.com/
