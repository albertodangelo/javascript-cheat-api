Transition

Um mit CSS eine Eigenschaft zu animieren kann mit transition bei der ursprünglichen Klassendefinition angegeben werden, mit der Eigenschaft, die verändert werden soll.

Zum Beispiel kann transform ausgeführt werden.

Sobald die entsprechende Klasse "mit Javascript" dem Element hinzugeführt wird, wird die Animation ausgeführt.

Hier ein Beispiel:

          .notification-container {
            position: absolute;
            bottom: -50px;
            transition: transform 0.3 ease-in-out;
          }
          
          .notification-container.show {
            transform: translateY(-50px);
          }
