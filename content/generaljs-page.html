Verschiedene wichtige Punkte

Selektoren

um ein einzelnes Element zu selektieren

            document.querySelector('.container')

um mehrere Elemente gleichzeitig zu selektieren (array)

            const container = document.querySelector('.container');
            const seats = document.querySelectorAll('.row .seat:not(occupied)');

Typen

typeof zeigt den Type einer Variable an (gut beim debuggen)

            console.log(typeof ticketPrice);

Anstelle von parseInt() zum Umwandlen des Types zu "Number" kann auch ein + verwendet werden 

            let ticketPrice = +movieSelect.value;

Array Methoden

Kopiert das Array "seats" und schaut, ob "seat" darin enthalten ist und speichert das Resultat im neuen Array "seatsIndex". Im Gegensatz zu "forEach", das eigentlich nur durch ein Array loopt, wird bei "map" mit "return" das Resultat in einem neuen Array gespeichert 

            const seatsIndex = [...selectedSeats].map( seat => {
                return [...seats].indexOf( seat );
            });

Math Funktionen

Aus einem Array zufällig einen Wert anzeigen

            const words = ["javascript", "html", "css", "server"];
            const selectedWord = words[Math.floor(Math.random() * words.length)];

Local Storage

localStorage benötigt ein "STRING value Paar" Objekt. Das "seatsIndex" Array wird daher in einen JSON String umgewandelt 

            localStorage.setItem('selectedSeats', JSON.stringify(seatsIndex));

das Gegenteil JSON.parse() macht aus einem String wieder ein Array

            const seatsLocalStorage = JSON.parse(localStorage.getItem('selectedSeats'));

Funktion für das Aktualisieren des Local Storage

            const updateSelectLocalStorage = (indexSelect, price) => {
              localStorage.setItem('indexSelect', indexSelect);
              localStorage.setItem('moviePrice', price);
          }

Event Listeners

Key Listeners: Beim drücken einer Taste wird mit 'keydown' ein Event ausgelöst. Jede Taste hat eine Zahl zugeordnet e.keyCode. Die Zahlen für die Buchstaben gehen von 65 bis 90 

    window.addEventListener("keydown", (e) => {
      console.log(e.keyCode);
    });

Backticks (Template Strings)

Mit Backticks können nicht nur Variablen sondern auch Funktionen in einem String dargestellt werden.
Beispiel: Ein "String" wird in ein "Array" umgewandelt split(), dann wird durch das Array geloopt map(), es werden erneut Backticks `` verwendet und erneut im "String" eine Funktion ausgeführt, dann wird das Array wieder in einen String umgewandelt join(). 

            const displayWord = () => {
              wordEl.innerHTML = `
                    ${selectedWord
                      .split("")
                      .map(
                        (letter) =>
                          `
                                
                                    ${correctLetters.includes(letter) ? letter : ""}
                                
                                `
                      )
                      .join("")}
                `;
            };

Verschiedenes

Neu ES6 falls providedData fehlt, wird der Wert aus der Variable "data" verwendet 

            const updateDOM = (providedData = data) => {
            }

Mit der replace() Funktion können Teile aus einem String ersetzt werden.
Beispiel: Die Zeilenumbrüche werden aus ein String entfernt. 

  const innerWord = wordEl.innerText.replace(/\n/g, "");

Mit .style kann man einfach einem Element einen Inline Style hinzufügen.

    popup.style.display = "flex";

Timer: Mit der Timeout Funktion lässt sich eine Funktion nach einer gewissen Zeit ausführen. 

        setTimeout(() => { 
          notification.classList.remove("show"); 
        }, 2000);
